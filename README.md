<h2>ğŸ› ï¸ Installation Steps:</h2>

<p>1. clone project</p>

```
git clone https://github.com/mhmdialireza/laravel-portfolio.git
```

<p>2. install dependencies</p>

```
composer install
```

<p>3. migrate DB &amp; run seeder</p>

```
php artisan migrate --seed
```

<p>4. compiling assets</p>

```
npm run dev
```

<p>5. serve the app</p>

```
php artisan serve
```
